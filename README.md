## 项目概述

针对《战地三》（Battlefield 3）在VU服务器上高分辨率屏幕下的UI缩放问题，提供了一款修复启动器，旨在优化游戏界面显示，提升玩家体验。

## 背景与问题

《战地三》在高分辨率屏幕上存在UI缩放不当的问题，表现为字体过小和界面布局不合理。根据知乎文章- [解决战地3的高分屏UI缩放问题 - 知乎](https://zhuanlan.zhihu.com/p/8621793244?utm_id=0)的分析，该问题源于游戏在分辨率超过1280x720时无法正确进行UI缩放，这是由游戏内部的硬编码值所导致。原文作者通过调试器定位了相关内存地址，并利用dll提出了一种解决方案，但该方案仅适用于官方服务器。

在原文分析的基础上，针对VU服务启动器通过监听进程然后修改内存的方式将原文作者提供的修复方案适配到VU服务器环境中

## 使用指南

1. **获取启动器**：
    - 克隆本项目仓库。
    - 直接运行或者打包`bf3-UIfix ver1.2.py`文件。
    - 或从本项目Release页面直接下载已编译完成的UI修复启动器。
2. **安装启动器**：
    - 解压缩下载的文件。
    - 将启动器程序放置于VU客户端的根目录下（与`vu.exe`文件同一层级）。
3. **配置启动器**：
    - 为避免杀毒软件误杀或权限问题，请以管理员权限运行启动器。
    - 将启动器添加至杀毒软件的白名单中，以确保启动成功和UI修复生效。
4. **启动游戏**：
    - 通过启动器启动游戏，UI修复功能将自动激活。

## 注意事项

- **兼容性提示**：本启动器专为战地三VU服务器设计，官方服务器用户请参考原文提供的解决方案- [解决战地3的高分屏UI缩放问题 - 知乎](https://zhuanlan.zhihu.com/p/8621793244?utm_id=0)。
- **反作弊说明**：在VU端带有pb反作弊的服务器中进行游玩二十分钟后，未出现被ban情况，但此不代表绝对安全，在本地bot服游玩未发现被ban的问题。

## 已知问题
